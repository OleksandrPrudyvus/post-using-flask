About web-application

If you have a 'user' role you have access to view all posts without the ability to edit and create posts.

Post This is the main tool of this site. In this site has the ability to watch, search, add, edit, and delete posts.

If you have a 'manager' role you have access to view all posts with the ability to edit and create posts.

If you have a 'admin' role you have access to view all posts with the ability to edit and create posts, create and delete users, tags and roles, but you can`t delete your account =(.
